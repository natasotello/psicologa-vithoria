🧠 Psicóloga Vithoria – Landing Page Profissional

Projeto desenvolvido para cliente real da área de Psicologia Clínica, com foco em presença digital estratégica e geração de agendamentos.

📌 Sobre o Projeto

Esta landing page foi criada com o objetivo de:

Apresentar os serviços de forma clara e humanizada

Gerar conexão e confiança com potenciais pacientes

Facilitar o agendamento de consultas

Direcionar o usuário para contato direto via WhatsApp

O design foi pensado para transmitir acolhimento, profissionalismo e credibilidade.

🚀 Tecnologias Utilizadas

HTML5

CSS3

JavaScript

Design Responsivo

Boas práticas de UI/UX

Estrutura semântica e otimização de performance

🎯 Funcionalidades

Layout 100% responsivo

Botão com redirecionamento direto para WhatsApp

Seção de perguntas frequentes com interação em JavaScript

Animações suaves para melhor experiência do usuário

Estrutura organizada para futura escalabilidade

💡 Estratégia de Desenvolvimento

O projeto foi estruturado priorizando:

Código limpo e organizado

Separação clara de responsabilidades (HTML, CSS e JS)

Facilidade de manutenção

Possibilidade de migração futura para domínio profissional
